# Proyecto de Desarrollo en Equipo

## Flujo de Trabajo

### Ramas de Desarrollo
- **`feature/equipoA`** - Desarrollo del Equipo A
- **`feature/equipoB`** - Desarrollo del Equipo B

### Proceso de Integración
1. Desarrollo paralelo en ambas ramas feature
2. Merge entre `feature/equipoA` y `feature/equipoB`
3. Revisión y aprobación del código integrado
4. Merge final a la rama **`main`**
